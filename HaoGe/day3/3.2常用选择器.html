css常用选择器
				1.全局选择器
					*{   //通配符
						
					}
				2.class选择器(类选择器)
					.类名{
						属性:属性值
					}
					class选择器一定要有点(.),英文中的点
					id/class命名规范:
						1)不能以数字开头,必须是字母开头
						2)可以有英文,下划线,数字连接组成
						3)尽量使用有意义的名字,尽量使用驼峰命名法
						4)严格区分大小写
				3.元素选择器
					就是选中相同的元素,然后给这个相同的元素给一个公共的样式
					标签名{
						属性:属性值
					}
				4.包含/后代选择器
					选中元素内部所有某一种元素,包含子元素以及其他后代元素
					选择器 选择器{
						属性:属性值
					}	
					父元素和后代元素必须用空格隔开使用,用来表示,所属关系
				5.子选择器
					查找某一标签的子集
					父元素>子元素{
						属性:属性值
					}
					父子关系,只有父元素的下一级生效
				6.伪类选择器
					选择器:hover{
						属性:属性值
					}
				7.群组选择器
					对于多个有公共样式,共同样式的元素设置样式
					选择器1,选择器2,选择器3{
						属性:属性值
					}
			选择器权重
				 !         10000
				行内       1000
				id选择器    100
				类选择器    10
				标签选择器  1
				伪类选择器  1
				全局选择器  0

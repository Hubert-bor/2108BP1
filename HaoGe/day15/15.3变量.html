变量
				1.什么是变量
					变量是计算机内存中存储数据的标识符,根据变量名称可以获取内存中存储的数据.
					javascript中,变量用于保存数据.
					通常来讲,使用方法,也就是使用规范
						var 变量名 = 值;
					javascript中,声明变量的意义就在于,用一个标识符来存储指定的数据.
				2.标识符
					标识符就是指一个自定义的名字.
					在js当中,变量,函数,数组等的名字,都可以称之为标识符.
					标识符命名的规范.
						1)变量名不能以数字开头;
						2)一般以字母或者下划线开头;
						3)不能使用中文;
						4)推荐使用驼峰命名法(相邻的两个字母,第二个字母首字母大写);
							比如: hotWater
						5)不能使用关键字或者保留字;
						6)区分大小写.
						   关键字   
							关键字: break  case  catch continue default (默认)  delete (删除) do else finally(总是) for function
									if、in、instanceof、new 、return 、switch、this、throw 、try、typeof 、var 、void 、while、with 等。
							保留字:
								abstract(抽象)、boolean、byte(字节)、char(字符)、class(类) 、const(常量) 、debugger(调试)、
								double(双精度浮点数)、enum(枚举) 、export (导出)、extends (继承) 、float(浮点数)
								implements(实现)、import(导入)、int(整型)、interface(接口)、long、package(包管理)、
								private(私有)、protected(保护)、public(公共)、static(静态)、super   
				3.声明变量
					var 变量名 = 值;
					同时声明多个变量,用逗号隔开;
					var a = 1,b =2,c=3;
				4.简单的拼接
					+号拼接
